//моргаем светодиодами

int led1Pin = 4;
int led2Pin = 5;
int led3Pin = 6;
int led4Pin = 7;

void setup() {    
  //инициализация выходов
  pinMode(led1Pin, OUTPUT);
  pinMode(led2Pin, OUTPUT);
  pinMode(led3Pin, OUTPUT);
  pinMode(led4Pin, OUTPUT);
}

void loop() {
  for (int ledPin=4;ledPin<8;ledPin++){//для пинов 4-7
    digitalWrite(ledPin, HIGH);//выключить светодиод
    delay(1000);// ждать 1 сек
    digitalWrite(ledPin, LOW);//включить светодиод
    delay(1000);//ждать 1 сек
  }
}

В первой строке мы объявили переменную (ledPin) и присвоили ей значение 4. И говорим Ardunio, что хотим менять в течении цикла  это значение от 4 до 7 (ledPin <8). ledPin++ говорит Ardunio, что переменную надо увеличивать на 1, каждый раз, когда повторяется цикл.
Затем идут строки с задержкой внутри цикла. Таким образом, с начала ledPin=4 и светодиод на 4 выводе включается и выключается, как прописано в задержке. После этого значение ledPin увеличивается на 1 и начинает работать вывод 5. Так происходит до тех пор, пока значение ledPin не станет равным 7

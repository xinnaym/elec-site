min()

Описание

Фунция min() вычисляет минимальное значение из двух чисел. Возвращаемое значение — меньшее из двух чисел.

Синтаксис

min(число 1, число2)

Параметры

число1, число2 — числа любого типа данных.

Пример

x = min(x, 60);  //переменной x присваивается меньшее из чисел x и 60

Реализация функции min() запрещает указывать другие функции в качестве параметров в скобках — это приведет к некорректным результатам:
Неправильно:
b = min(a++, 100);
Правильно:
a++;
b = min(a, 100);

max()

Описание

Фунция max() вычисляет максимальное значение из двух чисел. Возвращаемое значение — большее из двух чисел.

Синтаксис

max(число 1, число2)

Параметры

число1, число2 — числа любого типа данных.

Пример

переменной x присваивается меньшее из чисел x и 10

x = max(x, 10);

Реализация функции max() запрещает указывать другие функции в качестве параметров в скобках — это приведет к некорректным результатам:
Неправильно:
b = max(a--, 5);
Правильно:
a--;
b = max(a, 5);

abs()

Описание

Фунция abs() вычисляет абсолютную величину (модуль) числа.
Возвращаемое значение:

  • x — если x больше или равен 0.
  • -x — если x меньше 0.

Синтаксис

max(число)

Параметры

число — число любого типа данных.

Пример

переменной x присваивается абсолютная величина числа y.

x = abs(y);

Реализация функции abs() запрещает указывать другие функции в качестве параметров в скобках — это приведет к некорректным результатам:
Неправильно:
b = abs(x++);
Правильно:
x++;
b = abs(x);

constrain()

Описание

Фунция constrain() ограничивает значение переменной заданными пределами.
Возвращаемое значение:

  • x — если x лежит в пределах между a и b.
  • a — если x меньше a.
  • b — если x больше b.

Синтаксис

constrain(x, a, b)

Параметры

  • x — переменная, значение которой необходимо ограничить, любой тип данных.
  • a — нижний предел, любой тип данных.
  • b — верхний предел, любой тип данных.

Пример

числовые значения переменной v ограничены диапазоном от 10 до 20.

v = constrain(v, 10, 20);

map()

Синтаксис

map(value, fromLow, fromHigh, toLow, toHigh)

Описание

Фунция map() преобразовывает значение переменной из одного диапазона в другой. Т.е. значение переменной value, равное fromLow, будет преобразовано в число toLow, а значение fromHigh — в toHigh. Все промежуточные значения value масштабируются относительного нового диапазона [toLow; toHigh].

Возвращаемое значение: преобразованное значение.

  • x — если x лежит в пределах между a и b.
  • a — если x меньше a.
  • b — если x больше b.

Параметры

  • value — переменная, значение которой необходимо преобразовать.
  • fromLow — нижний предел текущего диапазона переменной value.
  • fromHigh — верхний предел текущего диапазона переменной value.
  • toLow — нижний предел нового диапазона переменной value.
  • toHigh — верхний предел нового диапазона переменной value.

Функция не ограничивает значение переменной заданными пределами, поскольку ее значения вне указанного диапазона иногда несут полезную информацию. Для ограничения диапазона необходимо использовать функцию constrain() либо до, либо после функции map().

Нижние пределы указываемых диапазонов (fromLow, toLow) численно могут быть больше верхних пределов (fromHigh, toHigh). В этом случае функция map() может использоваться для создания обратного диапазона чисел, например:

y = map(x, 1, 50, 50, 1);

Функция может обрабатывать отрицательные числа, например:

y = map(x, 1, 50, 50, -80);

Функция map() использует целочисленные вычисления, поэтому не возвращает дробных значений. При этом дробная часть числа просто отбрасывается, без округления или вычисления средних значений.

Пример

Преобразование аналогового значения в 8-битное число (от 0 до 255)

int v = analogRead(0);
v = map(v, 0, 1023, 0, 255);
analogWrite(9, v);

pow()

Описание

Фунция pow() вычисляет значение числа, возведенного в степень. Функция pow() может использоваться для возведения числа в дробную степень.

Синтаксис

pow(x, n)

Параметры

  • x — число, тип float.
  • n — показатель степени, в которую необходимо возвести число, тип float).

Возвращаемое значение: результат возведения в степень (тип double).

Пример

Числовые значения переменной a возводится в степень 5.

a = pow(a, 5);

sqrt()

Описание

Фунция sqrt() вычисляет квадратный корень числа.

Синтаксис

sqrt(x)

Параметры

  • x — число, любой тип данных.

Возвращаемое значение: квадратный корень числа, тип double.

Пример

Вычисление квадратного корня числового значения переменной a возводится в степень 5.

b = sqrt(a);

sq()

Описание

Фунция sq() вычисляет квадрат числа.

Синтаксис

sq(x)

Параметры

  • x — число, любой тип данных.

Возвращаемое значение: квадрат числа.

Пример

Вычисление квадрата числового значения переменной b.

x = sq(b);

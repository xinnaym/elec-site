C++ для Arduino

Программирование Arduino

После сборки схемы с Arduino нужно составить и загрузить программу управления. Программа – это последовательность действий, которые необходимо выполнить для решения поставленной задачи.

Программа для Arduino называется скетч (sketch). Для программирования Arduino используется упрощенная версия языка программирования C++ адаптированная для Arduino. В скетче Arduino, операторы программы (команды, инструкции), выполняются сверху вниз. Эта последвательность может быть изменено операторами управления.

Скетч состоит из двух основных функций:

  • setup() - это функция в которой задавются основные настройки: режимы работы выводов (вход или выход), подключение библиотек , инициализия переменных. Функция setup()запускается только один раз в течение скетча, когда стартует выполнение программы.
  • loop() - это основная функция, которая выполняется после setup(). Это функция будет выполняться бесконечно, пока вы не выключите питание.

Как и в других языках программирования, есть ряд правил написания исходного кода. Основные из них:

  • Ключевые слова, такие как if, for всегда записываются в нижнем регистре.
  • После каждого оператора необходимо ставить знак точки с запятой (;).
  • Фигурные с кобки {..} используются для обозначения программных блоков. Мы используем их для ограничения тела функции (см. ниже), циклов и условных операторов.
  • Перед объявлением функции необходимо указать тип данных, возвращаемый функцией или void если функция не возвращает значение.
  • Перед объявлением переменной необходимо указывать тип данных.
  • Хорошей привычкой является добавление комментариев к содержимому программы. Это помогает легко понять код.

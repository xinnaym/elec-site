Модуль датчика угла поворота (энкодер) KY-040

Модуль энкодера KY-040 — датчик, который преобразует угловое вращение ручки в цифровой сигнал.

Каждый энкодер имеет фиксированное количество циклов переключений на оборот, у KY-040 это число 30. Внутри энкодера находится медный диск с отверстиями соединенный с общей землей, назовем этот вывод «C» (рис. 2). Чем больше отверстия тем, точнее энкодер. Так же, есть два вывода, назовем их «А» и «B», с которых мы снимает сигнал, так же эти выводы подтянуты к 5 В с помощью резисторов на 10 кОм. Если контакты «А» и «B» не касаются вывода диска «B» на их выходе будет 5 В. Когда мы начнем вращать энкодер по часовой стрелке, вывод «А» коснется диска и на выходе получим «0 В», если продолжить вращать энкодер вывод «B» так же коснется диска и на выходе будет «0 В».

Таким образом, когда ручка поворачивается по часовой стрелке, сначала подключается вывод «А», а затем вывод «B». Когда ручка поворачивается против часовой стрелки, сначала подключается вывод «B», а затем штифт «А». Отслеживая, когда каждый вывод соединяется с землей и отсоединяется от нее, можно понять в каком направлении вращается ручка.

Технические характеристики KY-040:

  • Напряжение питания — 4,5...5,5 В.
  • Предельное напряжение на контактах — 5В.
  • Предельный ток через контакты — 10 мА.
  • Количество импульсов на один оборот — 24.
  • Формат выходного сигнала — квадратурный код 2 бит
  • Поворот — на: 360° в обе стороны.
  • Наличие кнопки — да.
  • Размеры — 26,2 ×18,7× 29 мм.
  • Температура эксплуатации — -30…70°C

Скетч: 

#define CLK 2 //вывод CLK энкодера подключен к выводу 2 Arduino
#define DT 3  //вывод DT энкодера подключен к выводу 3 Arduino
#define SW 4  //вывод SW энкодера подключен к выводу 4 Arduino
 
int counter = 0;                      // Создаем переменную counter
int currentStateCLK;                  // Создаем переменную currentStateCLK
int lastStateCLK;                     // Создаем переменную lastStateCLK
String currentDir ="";                // Создаем 
unsigned long lastButtonPress = 0;    // Создаем переменную lastBut
 
void setup() 
{
  pinMode(CLK, INPUT);       // Указываем вывод CLK как вход
  pinMode(DT, INPUT);        // Указываем вывод DT как вход
  pinMode(SW, INPUT_PULLUP); //Указываем вывод SW как вход и включаем подтягивающий резистор
  Serial.begin(9600);                 // Создаем последовательную связь 
  lastStateCLK = digitalRead(CLK);    // Считываем значение с CLK
}
void loop() 
{
  currentStateCLK = digitalRead(CLK); // Считываем значение с CLK
 
// Проверяем изменилось ли состояние CLK
  if (currentStateCLK != lastStateCLK  && currentStateCLK == 1){
    if (digitalRead(DT) != currentStateCLK) {
      counter --;
      currentDir ="CCW";
    } else {
      counter ++;
      currentDir ="CW";
    }
    Serial.print("Direction: ");
    Serial.print(currentDir);
    Serial.print(" | Counter: ");
    Serial.println(counter);
  }
 
  lastStateCLK = currentStateCLK;         // Запопоследнее состояние CLK
  int btnState = digitalRead(SW);         // Считываем состояние вывода SW
 
 
  if (btnState == LOW){                   // Если LOW, кнопка нажата
    if (millis() - lastButtonPress > 50){ // Если LOW в течении 50 мкс, кнопка нажата 
      Serial.println("Button pressed!");  // Вывод сообщения
    }
    lastButtonPress = millis();
  }
  delay(1);                               // Пауза
}

Транзистор BC337

Транзистор BC337 — биполярный высокочастотный транзистор общего назначения, n-p-n структуры. Предназначен для использования в схемах усиления и коммутации. Корпус TO-92.

Комплементарной парой для BC337 является транзистор BC327 c p-n-p структурой.

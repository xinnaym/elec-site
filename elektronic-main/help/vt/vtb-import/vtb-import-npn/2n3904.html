Транзистор 2N3904

2N3904 - биполярный маломощный высокочастотный транзистор n-p-n структуры. Применяется в малосигнальных схемах - входных каскадах усилителей и т.п. Комплементарной парой для 2N3904 является транзистор 2N3906 c p-n-p структурой.

Корпус: TO-92

Транзистор A1015

Транзистор A1015 — биполярный высокочастотный транзистор общего назначения, p-n-p структуры. Предназначен для использования в схемах усиления и коммутации.

Комплементарной парой для A1015 является транзистор C1815 c n-p-n структурой.

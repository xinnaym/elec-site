Транзистор A733

Транзисторы A733 - кремниевые, высокочастотные усилительные, структуры - p-n-p. Корпус пластиковый TO-93. Маркировка буквенно - цифровая.

Комплементарной парой для A733P является транзистор C945GR c n-p-n структурой.

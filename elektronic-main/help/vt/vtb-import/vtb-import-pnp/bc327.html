Транзистор BC327

Транзистор BC327 — биполярный высокочастотный транзистор общего назначения, p-n-p структуры. Предназначен для использования в схемах усиления и коммутации.

Комплементарной парой для BC327 является транзистор BC337 c n-p-n структурой.

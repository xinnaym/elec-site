Конденсатор К73-17

Конденсаторы К73-17 полиэтилентерефталатные защищенные предназначены для работы встроенных элементов внутри комплектных изделий в цепях постоянного, переменного тока и в импульсном режиме.

Вариант К73-17, К73-17а, К73-17г - для ручной сборки
Вариант К73-17б, К73-17в - для автоматизированной сборки (формованные выводы)
Вариант К73-17д - для замены конденсаторов МБМ

Проверка исправности симистора

Проверка стрелочным омметром:

Выполнить тестирование симистора мультиметром не удастся, потому что не хватит тока для открытия Поэтому нужно использовать стрелочный омметр или. Его силы тока будет достаточно для срабатывания.

Алгоритм проверки симистора стрелочнымй омметром:

  1. Подключаем щупы прибора к выводам T1 и T2 (A1 и A2).
  2. Устанавливаем кратность на омметре "×1".
  3. Проводим измерение, положительным результатом будет бесконечное сопротивление, в противном случае деталь «пробита».
  4. Кратковременно соединяем выводы T2 и G (управляющий). Сопротивление должно упасть примерно до 20-80 Ом.
  5. Меняем полярность и повторяем тест с пункта 3 по 4.

Проверяемый деталь не обязательно демонтировать, достаточно только отключить управляющий вывод.

В данным способом не всегда удается достоверно проверку, за исключением тестирования на «пробой», поэтому лучше собрать прибор для тестирования симисторов (рис. 1).

Алгоритм проверки:

  1. Устанавливаем переключатели в отжатое положение (соответствующее схеме).
  2. Нажимаем кнопку SB1: лампочка светится - симистор открылся.
  3. Нажимаем кнопку SB2: лампа гаснет - симистор закрылся.
  4. Нажимаем на кнопку переключателя SA1 и нажимаем кнопку SB1: лампа снова должна зажечься.
  5. Производим переключение SA2, нажимаем кнопку SB1, затем снова меняем положение SA2 и повторно жмем SB1. Индикатор включится, когда на затвор попадет минус.
